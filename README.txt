
# TCC - Online Course Platform (Minimal)

Arquivos:
- index.html — Login (usuário: admin / senha: 123 ou qualquer, redireciona para dashboard)
- dashboard.html — Meus cursos
- curso.html — Página do curso (vídeo + aulas + progresso)
- conta.html — Minha conta (perfil + atividades)
- style.css — Estilos gerais
- script.js — Menu retrátil e simulação do login
- logo.svg — Logo simples (pode substituir pela sua)

Como usar:
1. Abra `index.html` no navegador.
2. Faça login (admin / 123) → redireciona para `dashboard.html`.
3. Navegue pelos links do menu (☰).
